# Solstice Theme

A painterly VS Code theme suite featuring "Estival" (Light) inspired by Van Gogh and "Hibernal" (Dark) inspired by Monet. Soft, warm, and distraction-free.

> [!NOTE]
> This theme is a WIP and may have inconsistencies. Feedback is welcome!
> I do feel like I need a better name for it though...
